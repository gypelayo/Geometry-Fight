# Geometry-Fight
Geometry fight is a simple shoot-them up game but with a twist. 

Controls:
Right: D/Right_Key
LEft: A/Left_Key
Jump: W
Shoot: Mouse Click

Jumping to the walls will change the gravity to the vector that makes that wall the ground.
